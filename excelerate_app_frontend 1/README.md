# Excelerate Internship Support App

## Project Overview

The **Excelerate Internship Support App** is designed to help interns succeed globally by combining reflection, team building, and portfolio integration in one mobile platform.

---

## Vision

A mobile app that supports interns worldwide by:

- **Daily Pulse** → quick daily check‑ins for reflection and mentor feedback.
- **AI Team Builder** → automatically forming balanced teams using skills + time zones.
- **Portfolio Integration** → certificates and badges added to student profiles.

---

## Problem Statement

Interns face three main challenges:

1. Lack of professional communication and cultural adaptation.
2. Manual team formation is slow and unbalanced.
3. Mentors spend too much time fixing team problems.

---

## Target Users

- **Students** → log reflections, join teams, earn certificates.
- **Mentors** → review logs, give feedback, monitor team health.
- **Admins** → manage batches, generate teams, track analytics.

---

## Core Features (MVP – 1 Month)

- Role‑based login/signup.
- Daily Pulse check‑in (wins, blockers, culture notes).
- Mentor dashboard for quick feedback.
- AI team formation (skills + time zone).
- Team dashboard (members, chat, shared goals).
- Certificates/badges integrated into portfolio.

---

## Tech Stack

- **Frontend:** Flutter, Dart
- **Backend:** Supabase/PostgreSQL, Firebase Messaging
- **AI Engine:** Gemini API (resume parsing + skill matching)
- **Security:** JWT Auth, role‑based access

## User Journeys

**Learner Journey**

- Logs in with Google → completes Daily Pulse → enrolls in courses → submits assignments → earns certificates.

**Admin Journey**

- Logs in → reviews reflections → uses AI Team Builder → monitors dashboards → sends announcements → tracks certificates.

**Mentor Journey**

- Logs in → reviews Daily Pulse → gives feedback → monitors team health → supports interns.
